## CS330

## How do I approach designing software?
I approach designing software by first understanding the project requirements and breaking them into smaller tasks. For my OpenGL scene, I planned the layout before adding objects, textures, and lighting. This project improved my design skills by teaching me how to organize a 3D scene and use visual elements together.
I followed a step-by-step process, testing each addition as I went, and this approach can be applied to future software projects because it makes complex problems easier to manage. 

## How do I approach developing programs?
I develop programs by implementing one feature at a time and testing frequently to catch issues early. While creating my 3D scene, I used an iterative approach by making small adjustments to object placement, textures, materials, and lighting until everything worked correctly. 
Throughout the milestones, I became more confident with debugging and learned the value of building projects gradually instead of trying to complete everything at once.

## How can computer science help me in reaching my goals?
Computer science provides the technical and problem-solving skills needed to achieve my goal of becoming a software engineer. This project gave me experience with OpenGL, 3D transformations, textures, lighting, and shaders, which will be useful in future computer science courses. 
It also strengthened my programming and debugging skills, which are valuable in many professional software development careers, even outside of computer graphics.
